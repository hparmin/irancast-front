Bubble Chat با Clip-Path Responsive

این نمونه از دو روش برای ایجاد bubble chat استفاده می‌کند: clip-path و SVG mask. هر دو روش کاملاً responsive هستند و با تغییر سایز صفحه، شکل bubble حفظ می‌شود.

راهنمای استفاده:

  • کد بالا را در استایل عنصر تصویر خود قرار دهید
  • برای حفظ نسبت bubble، اطمینان حاصل کنید که عنصر حاوی تصویر دارای aspect ratio مربعی (۱:۱) باشد
  • مقادیر درصدی استفاده شده در clip-path باعث می‌شود شکل bubble در اندازه‌های مختلف responsive باشد
  • روش SVG mask گزینه بهتری برای پشتیبانی از مرورگرهای قدیمی‌تر است
  • می‌توانید رنگ و سایز bubble را با تغییر مقادیر در کد CSS شخصی‌سازی کنید
